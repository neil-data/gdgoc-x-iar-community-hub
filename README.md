## 🌐 Overview

**GDGOC × IAR** is a futuristic, interactive, and high-performance web experience built for the Google Developer Groups On Campus community at the **Institute of Advanced Research (IAR)**. The platform serves as a central hub for events, team showcases, schedules, roadmaps, and community engagement — all delivered through an immersive, visually rich interface.

Built on the modern **Next.js** ecosystem, the project leverages cutting-edge frontend technologies including **Three.js**, **Framer Motion**, **GSAP**, and **Tailwind CSS** to deliver a fast, scalable, and stunning developer experience.

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- **Node.js** ≥ 18.x
- **npm** ≥ 9.x or **yarn** ≥ 1.22.x or **pnpm** ≥ 8.x

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/gdgoc-iar/platform.git
   cd platform
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Set up environment variables**

   ```bash
   cp .env.example .env.local
   ```

   Update `.env.local` with your configuration values.

4. **Run the development server**

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

5. **Open in browser**

   Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
